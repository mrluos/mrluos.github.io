【正则】分组、捕获、预查

2016-04-08
javascript
js, 正则表达式

前言

上一篇正则表达式我们介绍了正则表达式的基本规则，这篇主要是记录关于正则表达式里的捕获。

捕获

在上一篇中介绍了使用小括号包围起来的表达式,会捕获匹配的内容到默认的分组中。

捕获是指把匹配的内容进行存储并编号，方面后续的使用。

非捕获

在实际实用中我们需要对字符进行分组匹配但是后续不需要操作匹配的内容，只是查找，如果我们使用（abc）这种模式明显就很浪费。
这里就引出只匹配不捕获的表达式：
(?:abc) ： 匹配abc.但是不对匹配的内容进行捕获。

2016-04-06
javascript
js, 正则表达式

前言

记录如何学习和使用正则表达式。

正则表达式

我对正则表达式的理解就是，用普通字符和特定字符组成一个“规则字符串”，用这个“规则字符串”来过滤操作的对象。正则表达式的规则不局限于语言，所以学习好正则能受用于各个平台。

字符

一个正则表达式可以由下面两种字符类型组合而成，也可以是单一的一种字符类型组成：
普通字符:包括大小写的字母和数字.
特定字符:在正则表达式中规定了一些代表某种意义的字符，也叫元字符。
