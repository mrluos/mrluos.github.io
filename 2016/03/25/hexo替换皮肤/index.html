hexo替换皮肤

发表于 2016-03-25   |   分类于 hexo   |  

前言

通过前面（hexo基本搭建 hexo发布到github）的折腾，现在已经把个人的站点搭建起来了，同时也能发布到github上，这篇文章记录一下如果看不顺眼站点的布局想要换一个该如何操作。

开始改变

step 1 选择主题

从hexo官方皮肤里找到你心仪的主题样式，这里我使用的NexT。

step 2 安装主题

打开NexT的github项目主页，这里提供2种安装的方法：
plan A
git clone ,打开git终端定位到你的站点，然后使用clone。

plan B
下载稳定版本的主题文件
1.从发布版本下载next主题Source Code (zip)，然后解压出来。
2.在你的站点的themes目录下新建一个next文件夹。
3.复制next文件夹内的所有文件到你的站点目录themes/next/里面。

step 3 配置

操作完成上面的步骤后，接着是配置使用下载的皮肤。
打开主站根目录下的_config.yml文件，在theme配置节点启用next.

然后重新生成和启动服务查看效果

至此就完成了皮肤的更换。
（完）
