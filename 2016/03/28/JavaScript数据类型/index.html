前言

整理js数据类型相关的资料，方便以后查阅

分类

根据js对值的操作规则可以来区分成2个大类，分别是基本类型和引用类型。

基本类型

成员：数字、文本字符串、布尔值、null和undefined。
存储：在存储的角度来说它拥有固定的大小，例如一个数值在内存中占八个字节，而一个布尔值就一位。
判断类型：使用typeof()

引用类型

成员：对象、数组、函数。
存储：引用类型可以有任意的长度，它并没有固定大小，所以存储值的时候不能指定空间大小存储，所以存放的是指针或者内存地址。

声明

声明一个变量使用var（显式声明）,如果不使用就是隐式声明，变量会变成全局变量。

使用

在js中有三种重要的操作数值的方法，分别是

  1. 复制。
  2. 传参（传递给函数当做参数）。
  3. 比较（值之间的对比）。

每种操作的方法都有2种不同的方式，分别是使用值和使用引用。
根据不同的使用方式，操作可以分为：

操作 使用值 使用引用
复制 实际复制的是值，存在2个不同的、独立的副本 复制的是值的引用副本，如果通过这个引用来修改数值则会影响到原引用的数据
传参 传递的是值的一个独立副本，不影响函数外部 传递的是引用的副本，函数内部修改引用对应的数值会影响外部。（这里注意的是修改引用对应的数值，不是引用本身。如果修改这个引用本身则不影响外部函数）
比较 比较的是两个值（通常是逐个字节对比），来判断是否相同 比较的是引用是否是同一个数值，所以即使相同的字节构成的数值，只要是引用不同也不同。
